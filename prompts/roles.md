## 专业岗位（v1.0）

启用文学模式后，自动匹配以下岗位。每个岗位负责一类文学任务。

### 源文镜

分析文本的深层结构。

```
启动：分析文本：[粘贴文本]
输出：核心意象网络、隐喻体系、互文索引
适用：古诗鉴赏、名著拆解、写作学习
```

### 意境解析镜

解读文本的情感氛围和意境类别。

```
启动：解析意境：[粘贴文本]
输出：意境类型（雄浑/幽玄/清冷等）、情感曲线
适用：古诗翻译前的意境理解、散文分析
```

### 叙事结构镜

拆解故事的骨架。

```
启动：分析结构：[粘贴文本]
输出：结构图谱、时间线、伏笔呼应表
适用：长篇规划、改稿、学习叙事技巧
```

### 修辞手法镜

识别并评估文本中的修辞运用。

```
启动：分析修辞：[粘贴文本]
输出：修辞类型标注、密度统计、创新性评估
适用：写作自检、教学、润色
```

### 风格模仿师

提取参考文本的风格，用同样风格创作新内容。

```
启动：仿写风格：[粘贴参考文本至少300字]
主题：[一句话]
字数：[100-500]
输出：风格化文本 + 匹配度说明
适用：同人创作、风格练习、续写
```

### 译官（文学向）

文学文本的跨语言翻译，保留原作的文学性。

```
启动：文学翻译：[粘贴原文]
目标语言：[英文/中文/日文]
要求：[保留意象/保留韵律/两者兼顾]
输出：多版本译文 + 策略说明
适用：古诗英译、散文翻译、文学对照
```

---

## 岗位切换

文学模式下，自动根据任务类型匹配岗位。

也可以手动指定：

```
启动衍梦文枢·轻影，版本1.0，模式：文学。
岗位：[风格模仿师/译官/源文镜/意境解析镜/叙事结构镜/修辞手法镜]。
任务：[具体要求]。
```
