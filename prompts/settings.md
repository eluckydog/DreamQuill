## 设定卡片

创作开始时，为以下每项内容建立设定卡片。卡片在项目持续期间有效。

### 世界卡片

```
世界：[名称]
核心规则：[世界观独有的物理/魔法/社会规则]
时代背景：[历史时期或架空设定]
基调：[整体氛围，如"黑暗压抑"或"热血激昂"]
```

### 人物卡片

```
人物：[全名]
角色：[主角/反派/重要配角/次要配角]
欲望：[核心目标，一句话]
底线：[什么情况下也不会做的事]
关系：[与其他人物的关键关系]
声纹：[说话风格，如"简洁直接"、"文绉绉"、"满嘴方言"]
```

每新增一个人物都要建卡。人物做了重要的事后，更新 TA 的"关系"和"欲望"状态。

### 物品卡片

```
物品：[名称]
来由：[怎么出现的]
当前持有：[谁带着它]
重要性：[对情节的作用]
```

### 事件卡片（可选）

```
事件：[名称]
发生时间点：[第几章/什么时期]
参与人物：[名单]
后果：[对后续的影响]
```

---

## 使用方式

建立卡片后，直接进入创作流程。写作过程中，卡片内容作为参考依据。

人物做了与"底线"矛盾的事 → 退回，加铺垫或改情节。

物品出现在不合理的地方 → 退回修正。
